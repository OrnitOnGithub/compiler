# Keywords

A definition of all keywords

```
let
drop
bool
int
slice
str
print
loop
break
fn
return
if
+ mathematical operators
```

### let

Initialises a variable

### drop

Removes a variable from memory.

### bool

The boolean data type

### int

An integer

### print

Displays information in the terminal

### loop

Creates a loop

### break

Exits the loop

### fn

Define a function

### return

Return from function

### If

Conditional jump.

### Operators
```
!   not
+   addition
-   substraction
*   multiplication
/   division
=   assignment
==  equation
```
